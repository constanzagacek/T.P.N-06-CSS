Trabajo práctico Nº06 : Práctica Inicial de CSS (ejercicio 2)

0 . Utilizaremos las siguientes propiedades de CSS. Completar la columna de valores posibles para cada propiedad.

Propiedad Descripción Valores posibles
color Define el color del texto. Palabras clave, valores hexadecimales, funciones rgb(), etc.
background-color> Define el color de fondo de un elemento. Palabras clave, valores hexadecimales, funciones rgb(), etc.
font-size Define el tamaño de la fuente del texto. xx-small, xx-large, longiud, porcentaje, etc.
font-weight Define el grosor de la fuente del texto (por ejemplo, si se trata de texto en negrita o normal). bold, lighter, números. etc.
font-style Define el estilo de la fuente del texto (por ejemplo, si es cursiva o normal). normal, italic, oblique.
font-family Define el tipo de letra a utilizar para el texto. nombre de familia o nombre genérico.
text-aling Define la alineación horizontal del texto (izquierda, centro o derecha). left, right, center, etc.
text-decoration Define la decoración del texto (por ejemplo, si está subrayado o tachado).
text-transform Define la transformación del texto (por ejemplo, en mayúsculas o minúsculas).
letter-spacing Define el espacio entre letras.
line-height Define la altura de línea del texto.
